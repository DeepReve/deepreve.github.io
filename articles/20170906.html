如何配置一台机器学习主机。


CPU

英特尔酷睿i3-6100，双核四线程

内存类型：DDR4-1866/2133,DDR3L-1333/1600 @ 1.35V

最大内存通道数：2

PCI Express 配置：Up to 1x16, 2x8, 1x8+2x4（多显卡时得注意）

支持的插槽：FCLGA1151（和主板的CPU插槽一致）

 

盒装版带风扇，散热效果一般。之后换上了“采融（Prolimatech）Basic81 CPU散热器”，CPU温度稳定在60度以内。

CPU详细参数：https://ark.intel.com/zh-cn/products/90729/Intel-Core-i3-6100-Processor-3M-Cache-3_70-GHz

显卡

技嘉GeForce® GTX 1080 G1 Gaming 8G

显存大小：8 GB

显存类型：GDDR5X

显存位宽：256 bit

总线标准：PCI-E 3.0 x 16

建议电源：500W

 

散热很不错，低负载时风扇停转，高负载时显卡温度在60度左右。

显卡详细参数：http://www.gigabyte.cn/Graphics-Card/GV-N1080G1-GAMING-8GD#sp

主板

技嘉Z270X-UD3

CPU：支持LGA1151插槽第七代/第六代处理器：Intel® Core™ i7处理器 / Intel® Core™ i5处理器 / Intel® Core™ i3处理器 / Intel® Pentium®处理器 / Intel®Celeron®处理器

内存：支持DDR4 3866(O.C.) / 3800(O.C.)/ 3733(O.C.) / 3666(O.C.) / 3600(O.C.) / 3466(O.C.) / 3400(O.C.) / 3333(O.C.) /3300(O.C.) / 3200(O.C.) / 3000(O.C.) / 2800(O.C.) / 2666(O.C.) / 2400(O.C.) /2133 MHz

扩展槽：

·  1个PCI Express x16插槽，支持x16运行规格(PCIEX16)
   * 为发挥显卡最大性能，安装一张显卡时务必安装至PCIEX16插槽。

·  1个PCI Express x16插槽，支持x8运行规格(PCIEX8)
   * 由于PCIEX8插槽与PCIEX16插槽共享带宽，所以当PCIEX8插槽安装显卡时，PCIEX16插槽最高以x8带宽运行。

·  1个PCI Express x16插槽，支持x4运行规格(PCIEX4)

·  3个PCI Express x1插槽 (所有PCI Express插槽皆支持PCI Express 3.0)

储存装置接口：1个M.2插座(支持Socket 3，Mkey，type 2242/2260/2280/22110 SATA 及PCIe x4/x2 SSD)

 

主板详细参数：http://www.gigabyte.cn/Motherboard/GA-Z270X-UD3-rev-10#sp

电源

海韵G-650电源（80PLUS金牌/半模组）

额定功率：650W

效率：90%

硬盘

三星750 EVO 250G SATA3 固态硬盘

读写速度：连续读取不超过540MB/s，连续写入不超过520MB/s

 

已有1TB机械硬盘，用来备份数据。该技嘉主板支持读写速度飞快的M.2NVMe固态硬盘，预算充足的同学可以考虑。

硬盘详细参数：http://www.samsung.com/cn/memory-storage/750-evo-sata-3-2-5-inch-ssd/MZ-750120BCN/

内存

威刚DDR4 2133 8GB 台式机内存

主频：2133 MHz

两条组成双通道。

机箱

乔思伯U4 银色 ATX机箱

机箱比较紧凑，在前面板加装了两个风扇，离显卡挺近的。

显示器

戴尔U2412M 24英寸16:10

几年前购买的，带鼠标和键盘。后来鼠标坏了，新买了“罗技（Logitech）M100r 光电鼠标”，手感比原来的好很多。

工具

南旗22合1螺丝刀

去年买的，拆过iPhone、Mac和iPad。

 

主机费用近9K，显示器的价格很坚挺，还是1599¥。预算充足的同学，请考虑更好的配置，毕竟更快的处理速度，可以帮助早出成果。
